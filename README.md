# react-native-buttercrypt

React Native Crypto tools

## Installation

```sh
npm install react-native-buttercrypt
```

## Usage

```js
import Buttercrypt from "react-native-buttercrypt";

// ...

const result = await Buttercrypt.multiply(3, 7);
```
